# Urban Logistics (Nairobi) — Sendy Logistics Challenge

I could not download this automatically: it's hosted on Zindi, which
requires a logged-in account to access the Data tab and download files.
There is no public, unauthenticated API for Zindi datasets.

## Manual steps (~5 minutes)

1. Create a free account at zindi.africa if you don't have one.
2. Search "Sendy Logistics Challenge" on Zindi (or go to the competition
   page directly from Zindi's competition list).
3. Join the challenge, open the Data tab, and download `Train.csv` (and
   `Riders.csv`, `Test.csv` if you want them).
4. Put the extracted CSV(s) in this folder.

Target variable: `Time from Pickup to Arrival` (delivery time in seconds).
Features: distance, pickup/destination coordinates, time of day, weather
conditions, rider info — a real last-mile delivery problem in Nairobi,
Kenya, with a genuine mix of numeric and categorical fields.
